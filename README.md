# FixedGearShop_v4

>該專案是練習 Vue.js 前端電商應用( *VueCLI_v4* )

## DEMO

> https://albertnotes.github.io/FixedGearShop_v4/#/

## Introduction

* 前台
    * FixedGear 介紹、影片滾動播放
    * 產品類別介紹 - 以 Carousel 呈現
    * 購物車功能
    * 優惠卷套用
    * 訂單結帳
* 後台
    * 登入 Modal
    * 產品新增、刪除、修改
    * 訂單查看
    * 優惠卷新增、刪除、修改
